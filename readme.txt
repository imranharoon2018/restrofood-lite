== Description ==
Restrofood Lite is an online food ordering and delivery system plugin for WordPress. You can manage your restaurant & other food ordering stuff with Restrofood. It has plenty of amazing features. That helps you to build a successful online business.


<h4>Main Features of Restrofood Lite:</h4>
<ul>
	<li>Easy WooCommerce Food Order System</li>
	<li>Ajax Quick Product View</li>
	<li>Ajax Quick search</li>
	<li>Single Page Checkout System</li>
	<li>Ajax Category Filter</li>
	<li>Ajax Pagination</li>
	<li>Delivery type select option ( Delivery/pickup )</li>
	<li>Delivery/pickup time select option</li>
	<li>Easy settings options</li>
	<li>Compatibility with wordpress 6+</li>
	<li>Translation ready</li>
	<li>Ready for translation ( .pot files included )</li>
	<li>Free Updates</li>
	<li>Error free code</li>
	<li>Well Documented</li>
</ul>

== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "Restrofood Lite". Click on Install and then Activate the Plugin.

That's it, Restrofood Lite Plugin is now activated on your site!

**The hard way..**

Download "Restrofood Lite WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
